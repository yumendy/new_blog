一看这样一个列表里套列表的，然后每个列表里又是元组，想到的就是字符画。

一个列表就对应于一行。这样的话就好办多了。挨着打出来就可以了，小小的修正一下程序：


```python
import pickle

fp = open('banner.p','rb')
banner = pickle.load(fp)
fp.close()
for line in banner:
    s = ''
    for tmp in line:
        s += tmp[0] * tmp[1]
    print(s)
```
